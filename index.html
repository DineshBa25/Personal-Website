BudgetTC Web Application
This is a fully featured budgeting system that allows you to track your spending, set goals, and create budgets. It also allows you to track your investments in a basic manner and allows you to set up a calendar to keep track of important dates. It also has a financial education section that will help you learn about the basics of personal finance.
What this application can currently do
  • Create a budget with subcategories
  • Give you a visual overview of the budget
  • Set up a calendar
  • Track your investments (limited due to API costs)
  • Learn about personal finance through a custom curriculum
  • Calculate the amount of money you need to save with a smart, in-depth Roth 401/IRA investment calculator
  • Save all user budget data to a live connected realtime database, and all other data to Cloud Firestore(firebase)
What this will be able to do soon
  • Create a draft budget and allow you to revert to previous versions of the budget
  • Give you smart feedback for the allocations in the budget
  • Connect the calendar to a live url iCal feed that exports to other calendar apps
  • Provide you with a dashboard that shows you and overview of everything on one page
  • Setup automatic motivation quotes to be texted to you when you wake up (powered by Twillo)
